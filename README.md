# IME For Minority Language
`小语种输入法`，支持梵文、巴利文、西夏文、藏文、婆罗米文等。
支持平台：windows7，8，10以及UWP。

本文仅仅是对项目的介绍，`小语种输入法`为私有项目，所有权归企业。

因此仓库没有代码，也没有任何项目文件。

`小语种输入法`由作者独立设计和完成（包含视觉和交互设计）
